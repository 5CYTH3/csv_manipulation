CSV Manipulation

NOT SUITABLE FOR PRODUCTION!

Ce petit projet est une simple représentation graphique de plusieurs données concernant les NFT (Non-Fungible Tokens ou jetons non-fongibles), récoltées à partir d'un dataset .csv

Pour ce qui est des librairies utilisées, nous avons travaillé avec pandas, principalement pour récupérer les données du fichier à disposition. Nous avons également utilisé matplotlib pour gérer la création de graphes, le traitement des données et son interprétation.
